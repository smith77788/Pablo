📋 Команды бота — справочник
Для менеджеров
/start — приветствие и команды
/orders — последние 10 заявок
/new_orders — только новые заявки
/models — список моделей
/stats — статистика агентства
Для клиентов
/start — приветствие с кнопками
/status NM-XXXX — статус заявки
Любое сообщение — переслать менеджеру
/start NM-XXXX — привязать заявку
⚙️ Первичная настройка:
1. Создайте бота через @BotFather в Telegram
2. Скопируйте токен → файл .env → TELEGRAM_BOT_TOKEN
3. Узнайте свой Telegram ID через @userinfobot
4. Добавьте ID в .env → ADMIN_TELEGRAM_IDS (через запятую)
