# Search, Commands & Action Palette

## Global search

Search across:
- accounts
- bots
- channels
- groups
- ecosystems
- templates
- operations
- reports
- keywords
- competitors
- unmanaged assets

## Natural searches

Examples:
- crypto
- all RU channels
- bots without commands
- groups without rules
- unstable accounts
- channels missing avatar
- channels using old template
- assets created this week

## Action palette

Supports natural commands:
- update all crypto channels
- find unstable accounts
- apply template to RU channels
- check all groups without rules
- create regional network for Germany
- repeat last publishing wave
- show channels missing avatars
- fix outdated bot commands

## Command mode

Support:
- /status
- /check all
- /create network
- /create regional
- /publish wave
- /apply template
- /make like this
- /repeat failed
- /find unstable
- /import bots
- /report visibility
- /fix drift
- /show issues

Power users must be fast.
