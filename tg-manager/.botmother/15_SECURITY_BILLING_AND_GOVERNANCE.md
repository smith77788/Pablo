# Security, Billing & Governance

## Security requirements

Required:
- encrypted Telegram sessions
- encrypted bot tokens
- encrypted proxy credentials
- no secrets in frontend
- role-based access
- workspace isolation
- project isolation
- audit logs
- approval for risky operations
- preview before mass operations
- safe operational limits
- secure workers
- emergency stop
- clear action history

Roles:
- Owner
- Admin
- Operator
- Analyst
- Viewer
- Billing Manager

## Governance

Sensitive operations require:
- permission check
- preview
- confirmation
- audit log
- report
- retry/failure handling
