## Make Others Like This

Critical feature.

User selects:
- channel
- group
- bot
- ecosystem

Then selects:
“Make others like this”

Infragram compares:
- descriptions
- usernames
- avatars
- commands
- buttons
- rules
- welcome messages
- pinned posts
- templates
- tags
- ecosystem structure

Then:
- shows preview
- shows what can be copied
- shows what cannot be copied
- applies to selected targets
- reports successes/failures
- allows retry failed

## Drift Detection

Detect when assets no longer match templates:
- description changed
- avatar missing
- commands outdated
- rules missing
- invite link missing
- username differs from pattern
- account lost admin rights
- bot missing start message
- group missing welcome message
- buttons missing
- pinned post missing
- template mismatch

Actions:
- Fix selected
- Ignore
- Save current as new template
- Generate report
