Create full ecosystem in one flow:

Example:

- 20 channels
- 5 groups
- 10 bots

- naming pattern

- descriptions
- first posts

After creation:

## Regional / Global Presence

Infragram supports distributed regional infrastructure.

User can deploy presence across:
- countries
- regions
- cities
- languages
- local communities

Examples:
- Berlin
- Paris
- Madrid
- Warsaw
- Kyiv
- London

For each city/region:
- channel
- group
- bot
- localized description
- localized username pattern
- localized welcome message
- local keywords
- local publishing settings

## Regional Network flow

1. Select countries/cities.
2. Select structure: channels/groups/bots.
3. Select templates.
4. Select account pool.
5. Select language/localization.
6. Preview.
7. Confirm.
8. Execute with reports and retry failed.
