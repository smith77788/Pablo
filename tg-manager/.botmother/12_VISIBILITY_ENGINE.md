# Visibility Engine

## Features

Support:
- keywords
- keyword groups
- regional keyword groups
- city keyword groups
- position tracking
- ranking history
- visibility score
- volatility score
- competitor tracking
- competitor comparison
- alerts
- reports
- trends
- visibility recommendations

## Connect visibility to operations

If keyword drops:
- suggest checking metadata
- suggest comparing competitors
- suggest reviewing description
- suggest visibility report
- suggest template sync if relevant
- suggest checking publishing history

No abusive suggestions.

## Regional visibility

Support:
- city-level keywords
- country-level keywords
- language-level keywords
- regional reports
- regional comparisons
- trend tracking
