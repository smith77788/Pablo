## 1. Начало работы

Отправьте команду `/start` менеджер-боту.

**Что происходит:**
- Бот проверяет, являетесь ли вы администратором (ваш Telegram ID должен быть в переменной `ADMIN_IDS`).
- Если вы администратор — открывается главное меню.
- Если нет — бот сообщает об отсутствии доступа.

**Главное меню содержит:**
| Кнопка | Действие |
|--------|----------|
| 🤖 Мои боты | Список всех добавленных ботов |
| ➕ Добавить бота | Добавить нового бота по токену |
| 📥 Импорт ботов | Массовый импорт нескольких ботов |
| 📦 Массовые операции | Групповое редактирование всех ботов |
| 🌐 Управление сетью | Аналитика и настройки сети ботов |

## 2. Добавление и управление ботами

### Добавить бота
1. Нажмите **➕ Добавить бота**.
2. Введите токен бота (получить у [@BotFather](https://t.me/BotFather) командой `/newbot`).
3. Бот проверит токен через Telegram API и добавит бота в список.

**Формат токена:** `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`

### Список ботов
- Показывается постранично (по 8 ботов).
- Кнопки **◀️ / ▶️** для навигации.
- Нажмите на бота — откроется его меню управления.

### Меню бота
После выбора бота из списка вы видите основное меню этого бота:

| Кнопка | Раздел |
|--------|--------|
| ✏️ Профиль | Редактирование имени, описания, фото |
| 👥 Аудитория | Статистика подписчиков, блокировки, экспорт |
| 📢 Рассылка | Отправка сообщений всем пользователям |
| ⏰ Расписание | Запланированные рассылки |
| 🤖 Команды | Команды бота (/start, /help и др.) |
| 📝 Шаблоны | Сохранённые тексты для рассылок |
| 💬 Авто-ответы | Автоматические ответы на сообщения |
| 📨 Inbox | Просмотр входящих, ответы пользователям |
| 🔗 Цепочки | Воронки с отложенными сообщениями |
| 🌐 Вебхук | Информация о webhook |
| 📊 Статистика | Подробная аналитика |
| 🏷 CRM | Теги пользователей |
| ⚙️ Автоматизация | Правила триггер→действие |
| 📝 Заметка | Личные заметки о боте |
| 🧬 Swarm | Роевая маршрутизация |
| 🧪 A/B Тесты | Тестирование вариантов |
| 🔗 Диплинки | Реферальные ссылки |
| 🎯 Активность | Сегменты вовлечённости |
| 📈 SEO | Оценка и советы по SEO |
| 🗑 Удалить | Удаление бота из платформы |

### Удаление бота
1. Нажмите **🗑 Удалить**.
2. Подтвердите удаление кнопкой **✅ Да, удалить**.
3. Бот будет удалён из платформы (сам бот в Telegram не удаляется).

## 3. Редактирование профиля бота

Раздел **✏️ Профиль** позволяет изменять публичные данные бота через Telegram Bot API.

### Имя бота
1. Нажмите **✏️ Имя**.
2. Введите новое отображаемое имя (до 64 символов).
3. Изменение применяется немедленно.

### Описание бота
1. Нажмите **📝 Описание**.
2. Введите текст описания (отображается на странице бота, до 512 символов).

### Короткое описание
1. Нажмите **💬 Краткое описание**.
2. Введите краткое описание (отображается в чате при старте, до 120 символов).

### Фото профиля
1. Нажмите **🖼 Фото**.
2. Отправьте изображение.
3. Фото устанавливается как аватар бота.

Для удаления фото: **🗑 Удалить фото**.

### Обновление токена
Если вы получили новый токен у BotFather:
1. Нажмите **🔑 Обновить токен**.
2. Введите новый токен.
3. Бот переключится на новый токен.

### Проверка работоспособности
Нажмите **🏥 Проверить** — бот делает тестовый запрос к Telegram API и сообщает статус: ✅ работает или ❌ ошибка.

## 4. Аудитория

Раздел **👥 Аудитория** — управление подписчиками бота.

### Просмотр статистики
Нажмите **📊 Статистика** — показывает:
- Всего пользователей
- Активных за последние 7 / 30 дней
- Заблокировавших бота

### Сканирование обновлений
Нажмите **🔄 Сканировать** — принудительно сканирует свежие обновления Telegram для актуализации базы пользователей.

### Экспорт в CSV
1. Нажмите **📤 Экспорт CSV**.
2. Получите файл `audience_<bot_id>.csv` со списком всех пользователей.

**Поля в CSV:** user_id, username, first_name, last_name, is_blocked, joined_at.

### Сравнение аудиторий
1. Нажмите **⚖️ Сравнить**.
2. Выберите второй бот из списка.
3. Получите отчёт: общие пользователи (пересечение), уникальные для каждого бота.

### Отправка сообщения конкретному пользователю
1. Нажмите **✉️ Написать пользователю**.
2. Введите Telegram ID пользователя (числовой).
3. Введите текст сообщения (HTML поддерживается).
4. Сообщение отправляется напрямую этому пользователю.

### Блокировка / разблокировка
- **🚫 Заблокировать** — введите ID, пользователь помечается заблокированным (бот перестаёт ему отвечать).
- **✅ Разблокировать** — введите ID, снимает блокировку.

## 5. Рассылка

Раздел **📢 Рассылка** — массовая отправка сообщений подписчикам.

### Новая рассылка
1. Нажмите **✏️ Написать сообщение**.
2. Введите текст рассылки (поддерживается HTML: `<b>`, `<i>`, `<code>`, `<a href="">`).
   - Или отправьте фото с подписью.
3. Опционально нажмите **🔗 Добавить кнопку** — введите текст кнопки и URL.
4. Нажмите **📤 Отправить** для немедленной отправки, или **🔬 Тест** для отправки только себе.

### Подтверждение перед отправкой
Перед рассылкой показывается превью с числом получателей. Нажмите **✅ Подтвердить** для запуска.

### Сегментированная рассылка
Нажмите **🎯 По сегменту** — выберите группу получателей:
- **Все** — все пользователи
- **Активные (7 дней)** — писали боту за последнюю неделю
- **Активные (30 дней)** — писали за последний месяц
- **Неактивные** — не проявляли активность более 30 дней

### Рассылка из шаблона
1. Нажмите **📝 Из шаблона**.
2. Выберите сохранённый шаблон.
3. Подтвердите и запустите.

### История рассылок
Нажмите **📋 История** — список последних рассылок с результатами:
- Дата и время
- Отправлено / ошибок
- Статус: выполнено / отменено / в процессе

## 6. Расписание

Раздел **⏰ Расписание** — отложенные рассылки.

### Создать запланированную рассылку
1. Нажмите **➕ Создать**.
2. Введите текст сообщения (или выберите из шаблона через **📝 Из шаблона**).
3. Введите дату и время в формате `ДД.ММ.ГГГГ ЧЧ:ММ` (например: `25.12.2025 10:00`).
4. Рассылка запланирована — запустится автоматически в указанное время.

### Отмена запланированной рассылки
В меню расписания нажмите **❌ Отменить** рядом с нужной рассылкой.

**Важно:** время указывается по UTC. Учитывайте разницу с вашим часовым поясом.

## 7. Шаблоны сообщений

Раздел **📝 Шаблоны** — сохранение текстов для повторного использования.

### Создать шаблон
1. Нажмите **➕ Добавить**.
2. Введите название шаблона (для вашего удобства).
3. Введите текст шаблона (HTML поддерживается).

### Использовать шаблон
- В разделах **Рассылка** и **Расписание** есть кнопка **📝 Из шаблона**.
- Выберите шаблон из списка — текст автоматически подставится.

### Просмотр и удаление
- Нажмите на шаблон в списке — увидите полный текст.
- Кнопка **🗑 Удалить** — удаляет шаблон безвозвратно.

## 8. Команды бота

Раздел **🤖 Команды** — настройка команд бота (кнопка "Menu" в Telegram).

### Добавить одну команду
1. Нажмите **➕ Добавить команду**.
2. Введите команду и описание в формате:
   ```
   start - Начать работу с ботом
   ```
3. Команда добавится к существующим.

### Установить все команды сразу
1. Нажмите **📋 Установить все**.
2. Введите список команд — каждая команда на новой строке:
   ```
   start - Начать
   help - Помощь
   info - Информация
   ```
3. Все команды устанавливаются через Bot API.

### Удалить все команды
Нажмите **🗑 Удалить все** — меню команд в боте очищается.

## 9. Авто-ответы

Раздел **💬 Авто-ответы** — автоматические ответы бота на сообщения пользователей.

### Создать правило авто-ответа
1. Нажмите **➕ Добавить**.
2. Выберите триггер:
   - **👋 /start** — срабатывает когда пользователь запускает бота
   - **🔑 Ключевое слово** — введите слово или фразу (поиск без учёта регистра)
   - **📩 Любое сообщение** — срабатывает на каждое входящее сообщение
3. Введите текст ответа (HTML поддерживается).

### Управление правилами
- Список правил отображается в меню авто-ответов.
- Нажмите на правило — откроется детальное меню.
- **✅/❌ Вкл/Выкл** — включить или отключить правило без удаления.
- **🗑 Удалить** — полностью удалить правило.

### Копирование правил в другой бот
1. Нажмите **📋 Скопировать в...**.
2. Выберите бот-получатель.
3. Подтвердите — все правила скопируются в другой бот.

**Приоритет:** правило `/start` проверяется первым, затем ключевые слова (по порядку создания), затем "любое сообщение".

## 10. Inbox (реле-ответы)

Раздел **📨 Inbox** — просмотр входящих сообщений и ответы пользователям в режиме оператора.

### Включение Inbox
Нажмите **🔔 Включить** — бот начнёт пересылать входящие сообщения пользователей вам в менеджер-бота.

### Ответ пользователю
1. Вы получаете пересланное сообщение от пользователя.
2. Ответьте на него (reply в Telegram) — ваш ответ автоматически отправится пользователю через управляемого бота.

### Быстрые ответы
В меню сессии доступны кнопки быстрых шаблонных ответов.

### Закрытие сессии
Нажмите **✅ Закрыть сессию** — сессия с пользователем завершается.

**Примечание:** Inbox позволяет живому оператору отвечать пользователям через бота, не раскрывая личный аккаунт оператора.

## 11. Цепочки сообщений (Воронки)

Раздел **🔗 Цепочки** — автоматические последовательности сообщений с задержками (email-воронки для Telegram).

### Создать цепочку
1. Нажмите **➕ Создать**.
2. Введите название цепочки.
3. Выберите триггер запуска:
   - **👋 /start** — при первом запуске бота
   - **🔑 Ключевое слово** — введите ключевое слово
4. Цепочка создана — теперь добавьте шаги.

### Добавить шаг в цепочку
1. Откройте цепочку и нажмите **➕ Добавить шаг**.
2. Введите текст сообщения для этого шага.
3. Введите задержку перед отправкой в секундах (0 — немедленно, 3600 — через час).

Шаги отправляются последовательно с указанными задержками после подписки пользователя.

### Управление цепочкой
- **✅/❌ Вкл/Выкл** — включить или приостановить цепочку.
- **🗑 Удалить** — удалить цепочку и все её шаги.

### Рассылка подписчикам цепочки
1. Откройте цепочку и нажмите **📢 Рассылка подписчикам**.
2. Введите текст.
3. Сообщение отправится только тем, кто подписан на эту воронку.

### Копирование цепочек из другого бота
Нажмите **📋 Копировать из...** — перенесёт все цепочки из выбранного бота.

## 12. Статистика

Раздел **📊 Статистика** — подробная аналитика бота.

**Доступные метрики:**
- Всего пользователей
- Новые за сегодня / неделю / месяц
- Активных за 24 часа / 7 дней / 30 дней
- Сообщений получено сегодня
- Рассылок отправлено
- Среднее вовлечение (%)
- Заблокировавших бота

Статистика обновляется в реальном времени при каждом открытии раздела.

## 13. CRM — Теги

Раздел **🏷 CRM** — сегментация аудитории с помощью тегов.

### Создать тег
1. Нажмите **➕ Создать тег**.
2. Введите название тега (латиница, кириллица, цифры, знак `_`).
3. Тег создаётся и становится доступен для использования в автоматизации.

### Просмотр тега
Нажмите на тег из списка:
- Показывает сколько пользователей имеют этот тег.
- Кнопка **🗑 Удалить тег у всех** — убирает тег у всех пользователей.

### Назначение тегов
Теги назначаются автоматически через правила **Автоматизации** (действие "Добавить тег").

**Использование тегов:**
- Сегментированные рассылки (только пользователям с определённым тегом).
- Триггеры в автоматизации ("когда добавлен тег X — выполнить действие Y").

## 14. Автоматизация

Раздел **⚙️ Автоматизация** — правила "если ... то ...".

### Создать правило
1. Нажмите **➕ Добавить правило**.
2. **Шаг 1 — Триггер:** выберите событие-запускатель:
   - **📩 Сообщение получено** — на любое входящее сообщение
   - **👤 Новый пользователь** — при первом обращении к боту
   - **🔑 Ключевое слово** — введите ключевую фразу
   - **🏷 Тег добавлен** — когда пользователю присваивается указанный тег
3. **Шаг 2 — Действие:** выберите что должно произойти:
   - **💬 Отправить сообщение** — введите текст ответа
   - **🏷 Добавить тег** — введите название тега
   - **🗑 Удалить тег** — введите название тега для удаления
4. **Шаг 3 — Название:** введите имя правила для удобства.

### Управление правилами
- Активных правил / всего — показывается в шапке раздела.
- Нажмите на правило для просмотра деталей.
- **✅/❌ Включить/Отключить** — временно приостановить без удаления.
- **🗑 Удалить** — полностью удалить правило.

## 15. Заметка о боте

Раздел **📝 Заметка** — личные заметки, видимые только администратору.

1. Нажмите **✏️ Редактировать**.
2. Введите текст заметки (без ограничений по длине).
3. Заметка сохраняется и отображается в меню бота.

Используйте для хранения: токенов, паролей, инструкций, ссылок, описания назначения бота.

## 16. Swarm — Роевая система

Раздел **🧬 Swarm** — интеллектуальная маршрутизация пользователей между ботами.

### Роли ботов
Каждому боту можно назначить роль в экосистеме:
| Роль | Назначение |
|------|-----------|
| 🚪 Entry (Вход) | Первая точка контакта, привлечение |
| 🔄 Conversion (Конверсия) | Перевод в целевое действие |
| 💎 Retention (Удержание) | Удержание и возврат аудитории |
| 🤖 General (Общий) | Общего назначения |

### Системные режимы
| Режим | Описание |
|-------|---------|
| Manual | Ручное управление, swarm выключен |
| Assisted | Рекомендации, решение за оператором |
| Autopilot | Полностью автоматическая маршрутизация |
| Growth | Акцент на привлечение новой аудитории |
| Experiment | Режим A/B тестирования |
| Stability | Приоритет стабильности над экспериментами |

### Включение Swarm
Нажмите **🔄 Вкл/Выкл** — включает или отключает роевую маршрутизацию для этого бота.

### Статистика роутинга
Нажмите **📊 Статистика** — показывает количество переходов пользователей между ботами.

## 17. A/B Тесты

Раздел **🧪 A/B Тесты** — тестирование разных вариантов сообщений.

### Создать тест
1. Нажмите **➕ Создать**.
2. Введите название теста.
3. Выберите тип теста:
   - **👋 Start message** — тестирует приветственное сообщение при /start
   - **💬 Reply** — тестирует авто-ответ
4. Добавьте варианты (минимум 2):
   - Нажмите **➕ Добавить вариант**
   - Введите название варианта (A, B, ...)
   - Введите текст этого варианта

### Управление тестом
- **▶️ Запустить** — тест начинает показывать варианты разным пользователям поровну.
- **⏸ Пауза** — временная остановка.
- **▶️ Возобновить** — продолжить тест.

### Выбор победителя
1. Нажмите **🏆 Выбрать победителя**.
2. Выберите лучший вариант из списка с метриками.
3. Победивший вариант становится основным.

### Удаление теста
**🗑 Удалить** — тест и все данные удаляются.

## 19. Активность и вовлечённость

Раздел **🎯 Активность** — сегментация аудитории по степени активности.

|---------|---------|

Для реактивации: введите текст сообщения → бот разошлёт его целевому сегменту.

### Тепловая карта активности
Нажмите **🗺 Карта активности** — показывает по часам и дням недели, когда пользователи наиболее активны. Используйте для планирования времени рассылок.

### Топ пользователей
Нажмите **⭐ Топ активных** — список самых активных пользователей с числом сообщений.

### Автоматическая расстановка тегов
Нажмите **🏷 Авто-теги** — автоматически присваивает теги `hot`, `warm`, `cold`, `lost` пользователям по их активности. Полезно для последующей сегментации рассылок.

## 24. Управление сетью ботов

Раздел **🌐 Управление сетью** — глобальная аналитика и настройки всей сети ботов.

### Аналитика сети
Нажмите **📊 Аналитика** — суммарные метрики по всем ботам:
- Общее число пользователей в сети
- Активность по сети
- Распределение пользователей между ботами

### Кластеры ботов
Нажмите **🗂 Кластеры** — просмотр групп ботов.

**Управление кластером:**
- **🔄 Swarm Вкл/Выкл для всех** — включить/выключить роевую маршрутизацию для всего кластера.
- **📌 Роль для всех** — назначить роль всем ботам кластера.
- **➕ Добавить бот в кластер** — привязать бот к этому кластеру.

### Рейтинг ботов
Нажмите **🏆 Рейтинг** — боты отсортированы по числу активных пользователей.

### Веса маршрутизации
Нажмите **⚖️ Веса** — настройка вероятности направления нового пользователя в тот или иной бот.

1. Нажмите **✏️ Изменить вес** рядом с ботом.
2. Введите число от 1 до 100 (чем выше — тем больше пользователей получит этот бот).

1. Нажмите **📢 Рассылка по сети**.
2. Введите текст сообщения.
3. Подтвердите — сообщение разошлётся всем пользователям всех ботов в сети.

**Осторожно:** охват равен сумме аудитории всех ботов.

### Клонирование настроек
1. Нажмите **📋 Клонировать настройки**.
2. Выберите бот-источник.
3. Выберите бот-получатель.
4. Подтвердите — команды, авто-ответы и настройки скопируются.

### Пересечение аудиторий
Нажмите **🔍 Пересечения** — показывает пользователей, которые есть одновременно в нескольких ботах сети. Помогает избежать дублирования рассылок.

## Горячие клавиши и подсказки

- **◀️ Назад** — всегда возвращает на предыдущий уровень меню.
- **HTML в тексте:** используйте `<b>жирный</b>`, `<i>курсив</i>`, `<code>моноширинный</code>`, `<a href="url">ссылка</a>`.
- **Отмена FSM:** если бот ждёт ввода и вы хотите отменить — отправьте `/start` для сброса состояния.
- **Лимиты Telegram:** имя бота — до 64 символов, описание — до 512, краткое — до 120.

## Часто задаваемые вопросы

**Почему бот не отвечает пользователям?**
1. Проверьте авто-ответы в разделе **💬 Авто-ответы** — должно быть хотя бы одно активное правило.
2. Убедитесь, что пользователь не заблокирован.
3. Проверьте вебхук — возможно, установлен конфликтующий URL.

**Как узнать Telegram ID пользователя?**
Попросите пользователя написать боту @userinfobot — он покажет его ID.

**Рассылка зависла на 0%?**
Возможно, у бота нет подписчиков, или все пользователи заблокировали бота. Проверьте раздел **👥 Аудитория**.

**Как удалить бота из менеджера?**
Меню бота → **🗑 Удалить** → подтвердить. Это удалит бота из платформы, но не удалит аккаунт бота в Telegram.
